
### 📘 개요


상용 서비스 틱톡을 클로닝한 프로젝트로 FireBase 연동 제작

### 🏃🏻‍♂️ 프로젝트 기한

2023.06 ~ 2023.07

### 🧑🏻‍💻 참여 팀원

혼자 진행

### 🛠️ 기술 스택

- Flutter
- riverpod
- FireBase
- mvvm + repo

### 🙋🏻 주요 기능 & 담당 역할

- mvvm + repo 디자인패턴을 활용한 설계
- riverpod을 활용한 상태관리
- Firebase 를 통한 회원가입, 로그인 / 영상저장 / APNS 연동
- Github 소셜 로그인 연동
